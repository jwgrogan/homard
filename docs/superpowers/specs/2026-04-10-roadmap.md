# Homard Roadmap

## v1.0 — Daily Driver (DONE)
- [x] Rust daemon with ReAct agent loop
- [x] Codex app-server backend (sub-second chat)
- [x] Claude CLI backend
- [x] Telegram integration with username allowlist
- [x] macOS tray app (Tauri)
- [x] Server mode (launchd KeepAlive)
- [x] Identity files (SOUL, USER, MEMORY, AGENTS, etc.)
- [x] Security: permissions, sandbox, auth
- [x] `homard setup` interactive wizard
- [x] Homebrew formula

## v1.1 — Multi-Machine (NEXT)
- [ ] Windows support (.exe + tray)
- [ ] Separate Telegram bots per machine
- [ ] Cron management UI in tray app
- [ ] Activity tab redesign (session manager)
- [ ] `homard setup` auto-installs codex/claude CLI + login

## v2.0 — Telegram Mini App
- [ ] Web app inside Telegram (no native app needed)
- [ ] Multi-machine dispatch ("send to Mac / Windows")
- [ ] Supabase backend for machine registry
- [ ] Identity file sync across machines
- [ ] Cron management UI in Mini App

## v2.1 — Smart Assistant
- [ ] MCP integration (Gmail, Calendar, Slack via MCP servers)
- [ ] Agent installs MCP servers on demand ("connect my Gmail")
- [ ] Todo lists and reminders
- [ ] Proactive notifications ("your PR has been open 3 days")
- [ ] HEARTBEAT.md auto-sync with cron

## v3.0 — Platform
- [ ] Multi-user support (team Telegram group)
- [ ] Shared memory across team members
- [ ] Agent marketplace (custom SOUL/AGENTS templates)
- [ ] Self-hosted Supabase option
